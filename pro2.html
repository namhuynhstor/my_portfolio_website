Brief about the project

The project will be based on the area of Southeast Asian, called S.E.A

In this project, I will use SQL Server to explore data on Covid-19 in Southeast Asian nations (Brunei, Myanmar, Cambodia, Timor-Leste, Indonesia, Laos, Malaysia, Philippines, Singapore, Thailand, and Vietnam). Following the exploration, I will visualize the data in Tableau.

The project will be divided into two sections, the first of which will be Data Exploration and the second of which will be Data Visualization.

About dataset

Our World in Data contributed the data. From early 2020 until the present, the dataset provides the worldwide coronavirus pandemic report. All data for this study is maintained in two tables: 'CovidDeaths' and 'CovidVaccinations' with the former including information about infection and fatalities and the latter containing information on vaccines.

Please keep in mind that the dataset is updated everyday. This project's data is current as of June 23, 2023.

Process Planning

Step 1: Download full data (.csv) from Our World in Data.
Step 2: Run the csv file with Excel and perform data sorting and splitting to 'CovidDeaths' and 'CovidVaccinations'.
Step 3: Import those in SQL Server do ETL(Extract, Transform, Load).
Step 4: Download Tableau Public (Free) or Tableau Desktop (14 days trial) to perform Data Analysis.
Step 5: Connect Tableau with SQL Server database or Excel database.
Step 6: Save the file as (.twb or .twbx).

1. Data Exploration with SQL Server

The exploration will focus mainly on these topics:

  • Cases and deaths of each countries in SEA.
  • Vaccinations of each countries in SEA.
  • Comparison cases and deaths between Mainland Southeast Asia and Maritime Southeast Asia.

    ** Create Temp Table that include only countries in Southeast Asia

    -- DROP the any table named CovidSEA
DROP TABLE IF EXISTS CovidSEA

-- CREATE the table (CovidSEA)
CREATE TABLE CovidSEA
   (
	location nvarchar(255), 
	region nvarchar(255),
	date datetime,
	population numeric,
	new_cases numeric,
	new_deaths numeric,
	doses_administered numeric,
	new_vaccinations numeric,
	people_vaccinated numeric,
	people_fully_vaccinated numeric
   )
 
-- INSERT to CovidSEA
INSERT INTO CovidSEA
SELECT	dea.location,
	CASE WHEN dea.location IN ('Cambodia', 'Laos','Myanmar','Thailand','Vietnam') THEN 'mainland'
		ELSE 'maritime'
		END AS region,
	dea.date_new,
	dea.population,
	dea.new_cases,
	dea.new_deaths,
	SUM(CAST(vac.new_vaccinations as bigint)) OVER(Partition by dea.location Order by dea.location, dea.date) AS doses_administered,
	vac.new_vaccinations,
	vac.people_vaccinated,
	vac.people_fully_vaccinated
FROM 	CovidDeaths dea
JOIN 	CovidVaccinations vac
  ON 	dea.location = vac.location
  AND	dea.date_new = vac.date_new
WHERE 	dea.continent IS NOT NULL
  AND 	dea.location IN ('Brunei','Cambodia','Indonesia','Laos','Malaysia','Myanmar','Philippines','Singapore','Thailand','Timor','Vietnam')

    I've built a temporary database named CovidSEA that contains only SEA countries and columns that I'll use in the investigation by linking the CovidDeaths and CovidVaccinations tables together.

    The CASE WHEN method was also used to generate a new column called'region', which would designate whether the nation is located in Mainland SEA or Maritime SEA.

    Furthermore, I use the Window Function (PARTITION BY clause) to split the number of vaccines into partitions and then use the SUM() function on each subset of partitioned data, which in this case is partitioned by location column, to get the total number of vaccinations for each country.

    ** Overall Infection and Death Numbers in Southeast Asia

    SELECT 	SUM(DISTINCT(population)) as TotalPopulation,
	SUM(new_cases) as TotalCases,
	SUM(new_deaths) as TotalDeaths,
	SUM(new_deaths)/SUM(new_cases)*100 as DeathPercentage,
	SUM(new_cases)/SUM(DISTINCT(population))*100 as PercentPopulationInfected,
	SUM(new_deaths)/SUM(DISTINCT(population))*100 as PercentPopulationDeath
FROM CovidSEA

    → There are 36,284,653 cases with a total of 368,313 fatalities, equating to a fatality percentage of 5.33

    SELECT	location,
	MAX(DISTINCT(population)) as population,

    → Vietnam ranks highest in terms of the total number of COVID-19 infections recorded (11,619,322), whereas Brunei has the highest proportion of its population infected with the virus (68.77%).

    ** Letâ€™s deep down to vaccinations in SEA

    WITH VacSEA(TotalPopulation, TotalDoses, TotalPartlyVaccinated, TotalFullyVaccinated) AS
(
SELECT	SUM(DISTINCT population) AS TotalPopulation,
    	SUM(new_vaccinations) AS TotalDoses,
	(
	  SELECT  SUM(val1)
	  FROM	  (
		     SELECT MAX(people_vaccinated) AS val1
		     FROM   CovidSEA
		     GROUP BY location
		  ) a
	) AS TotalPartlyVaccinated,

        (
          SELECT SUM(val2)
          FROM	  (
		     SELECT MAX(people_fully_vaccinated) AS val2
		     FROM   CovidSEA
		     GROUP BY location
		 ) b
	) AS TotalFullyVaccinated
FROM CovidSEA
)

SELECT	TotalPopulation,
    	TotalPartlyVaccinated,
    	TotalFullyVaccinated,
    	TotalPartlyVaccinated / TotalPopulation * 100 AS PercentagePartlyVaccinated,
    	TotalFullyVaccinated / TotalPopulation * 100 AS PercentageFullyVaccinated
FROM VacSEA;

    → By 23 June 2023, a significant milestone had been achieved in Southeast Asia, with 78% of the population having received at least one dose of the COVID-19 vaccine, while 70% of the population had completed their vaccination journey. It is worth mentioning that the utilization of the WITH clause has played a crucial role in organizing and streamlining lengthy and intricate hierarchical queries, contributing to improved efficiency and readability.

    ** Break vaccinations down by countries

    WITH VacSEAbyCountry (location, population, TotalDoses, PartlyVaccinated, FullyVaccinated, PartlyVaccinatedPercentage, FullyVaccinatedPercentage) AS
(
SELECT	location,
	MAX(DISTINCT(population)) as Population,
  	SUM(new_vaccinations) as TotalDoses,
  	MAX(people_vaccinated) as PartlyVaccinated,
  	MAX(people_fully_vaccinated) as FullyVaccinated,
  	MAX(people_vaccinated)/MAX(population)*100 as PartlyVaccinatedPercentage,
  	MAX(people_fully_vaccinated)/MAX(population)*100 as FullyVaccinatedPercentage
FROM 	CovidSEA
GROUP BY location)
	
SELECT	*
FROM 	VacSEAbyCountry
ORDER BY location;

    → Interestingly, Brunei has the highest vaccination rate, but from the previous queries, Brunei also has second highest percent of infected population. This might be because of Brunei has a relatively small population, which allows for easier logistics and distribution of vaccines. The country's well-established healthcare infrastructure and effective communication channels have also facilitated the vaccination process.
